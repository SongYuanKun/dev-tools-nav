# dev-tools-nav 使用说明

## 1. 项目定位

`dev-tools-nav` 是一个纯静态的开发者工具导航站，核心能力包括分类筛选、搜索、收藏、最近访问、暗色模式与隐藏彩蛋分类。

## 2. 本地运行

在仓库根目录执行：

```bash
python3 -m http.server 8080
```

然后访问：`http://127.0.0.1:8080/index.html`

## 3. 主要功能

- 分类筛选：点击顶部分类按钮切换工具集合。
- 搜索：在搜索框输入关键词，按名称/描述/标签实时过滤。
- 收藏：卡片右上角点击 `☆`/`★` 加入或取消收藏。
- 最近访问：点击“访问/详情”会自动记录到“最近访问”标签页。
- 主题切换：右上角按钮在亮色/暗色之间切换并持久化保存。

## 4. 彩蛋解锁方式

隐藏分类“激活工具”支持以下方式解锁：

- URL 参数：`?devtools2024=unlock`
- Logo 连击
- Konami 代码
- `Ctrl/Cmd + Shift + A`
- 页脚问号连续点击

## 5. 本次修复说明（稳定性）

修复文件：`js/main.js`

问题：在隐私模式或受限浏览器环境下，直接访问 `localStorage` 可能抛出异常，导致页面初始化流程中断。  
处理：新增 `SafeStorage` 封装并将主题与彩蛋相关存储访问改为安全读写，保证存储不可用时页面仍可正常加载与交互。

## 6. 验证记录

- 语法检查：`node --check js/main.js && node --check js/base.js` 通过。
- 页面可访问性冒烟：本地启动静态服务后，`curl -I /index.html` 返回 `HTTP/1.0 200 OK`。
