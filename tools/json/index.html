FAQ

常见问题

输入的 JSON 会上传到服务器吗？

不会。输入、转换和对比都只在当前浏览器中处理。

宽松解析会修改哪些内容？

宽松解析只接受注释和尾逗号；单引号转换必须由你明确选择安全修复。

JSON 与 YAML 转换是否完全无损？

工具会拒绝 JSON 无法无损表示的 YAML 类型、复杂键、多文档和别名。

如何快速定位 JSON 错误？

错误横幅会显示行列；选择定位即可把光标移动到首个错误字符。
