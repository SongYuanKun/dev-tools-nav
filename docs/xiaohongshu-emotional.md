# 为什么一个程序员，会花3天做一个导航站？

## 起因很简单

作为一个程序员，浏览器书签栏里有几百个网址。
每次找一个工具都要翻半天，有些甚至不知道当初为什么收藏。

某天在一个技术群里看到有人问："有没有好用的截图工具推荐？"
群里刷屏推荐，五花八门。

我就想：**干脆做个导航站，把常用的工具都整理出来。**

## 做着做着就认真了

第一天，我想：这不就是个静态页面吗？两小时搞定。

结果做着做着，想法越来越多：
- 要不要加搜索？
- 要不要做详情页？
- 要不要支持暗色模式？
- 要不要加个彩蛋？

每个"要不要"，后面都是几个小时的实现。

**原来"简单"的事情，认真做起来都不简单。**

## 关于那个彩蛋系统

花了一天时间做了个彩蛋系统：5 种解锁方式，撒花动画，本地存储。

**为什么花这么多时间？**

因为想让这个网站有趣一点。

程序员的快乐很简单：
- 一次成功的代码运行
- 一个完美的解决方案
- 或者一个藏着的小惊喜

## 目前的情况

老实说，**这个项目现在还没什么人看到。**

但我还是会继续更新。

为什么？

**1. 这是一个学习过程**
从想法到实现，从功能到体验，每一步都在学习。

**2. 整理自己的知识**
把常用工具整理出来，对自己也是一次知识梳理。

**3. 积累作品**
作为独立开发的练手项目，积累经验和作品。

**4. 也许能帮到别人**
虽然现在没什么人看，但也许哪天就能帮到某个找工具的程序员。

## 一些数据

- 60+ 次提交
- 2000+ 行代码
- 61 款工具收录
- GitHub Actions 自动化运维

## 下一步计划

- 继续补充工具内容
- 优化用户体验
- 学习 SEO 优化，让更多人看到
- 考虑添加用户提交工具的功能

## 写在最后

如果你也是程序员，也有想法想做点什么：
**那就先做出来，别想太多。**

完美主义是进步的大敌。
先完成，再完美。

也许一开始没什么人看，
但只要你坚持更新，总会有收获。
