我的故事

从2017年开始写代码，我的职业主线一直是Java后端开发。技术栈以 Spring Boot + MySQL 为核心，日常和 MyBatis、Redis、消息队列打交道。但我从不想做一个"只会写业务代码的程序员"——技术本身，才是让我兴奋的东西。

于是我开始向外探索：从深入 MySQL 主从复制原理，到剖析 LongAdder 的并发优化思路；从搭建 Jenkins + SonarQube 的 CI/CD 流程，到研究 Git 与代码托管的最佳实践。每一篇博客、每一个踩坑记录，都是我对技术边界的试探。

现在，我开始走向全栈与独立开发——这个工具箱就是我的第一个独立作品。它收录了我日常真正在用的工具，也是我从前端到部署、从产品到运维的完整实践。我不想只做需求的执行者，我想用技术，做自己的产品。
