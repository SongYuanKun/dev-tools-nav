☕ JRebel 激活

JRebel 是 Java 热部署工具，修改代码后无需重启应用即可生效，大幅节省开发时间。支持 2023.4.0+ 版本，通过 License Server 激活。

激活步骤

  1. 打开 IDE（IntelliJ IDEA / Eclipse 等），进入 JRebel → Settings / Configuration。
  2. 选择 Activate JRebel，选择 Connect to a license server。
  3. 在 License server URL 中粘贴上方地址（页面加载后显示）。
  4. 点击 Activate 完成激活。

一键无限试用脚本（bat）

在 Windows 下可运行以下脚本重置试用（请勿用于商业用途）。

@echo off
echo 正在重置 JRebel 试用...
set JRebel_HOME=%USERPROFILE%\.jrebel
if exist "%JRebel_HOME%" (
  rd /s /q "%JRebel_HOME%"
  echo 已删除 %JRebel_HOME%
)
echo 完成。请重启 IDE 后重新激活。
