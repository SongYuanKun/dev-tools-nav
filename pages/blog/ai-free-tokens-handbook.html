AI 免费 Token / 额度薅羊毛手册

整理于 2026 年 3 月 作者 Koen 约 12 分钟阅读
AI API 新手向

这篇写得比较长，是故意啰嗦的：方便完全不懂代码、也没做过网站的读者，也能大致明白「免费体验 AI」这件事在说什么、坑在哪里。如果你自己会做网站、会调接口，可以跳着看小标题。

各家的免费额度、要不要绑信用卡、能用哪些「型号」的 AI，经常会变；下面只能当指路用的笔记。真要注册、尤其是要绑卡之前，一定要自己打开官网，看当前页面上的说明，别只信转载文章里的数字。

给想少花钱、又想试试「把 AI 接进自己的东西里」的人看的——「自己的东西」可以是一个小网站、一个小程序、一个作业演示，甚至只是你想先免费玩玩、感受一下不同平台差别。不一定要马上充很多钱，很多平台都有免费层或者送一点试用金。

如果你要把这篇转到别的群、公众号里发，文末我留了自己的工具导航链接；你发的时候换成你自己的链接就行，免得给别人导流。

读前扫盲：三个词先搞懂，后面就不晕

1. API（可以粗俗理解成「饭店点菜窗口」）
你不用管厨房怎么炒菜，只要在窗口按菜单下单，厨房做好了把菜递出来。对 AI 来说：你的程序（或网站）往对方的「窗口」发一段文字，对方把 AI 的回答发回来。很多「免费 token」说的就是：这个窗口允许你免费试多少次、或免费试多少字。

2. Token（代币 / 字数的一种计费方式）
你不用算得很精确，只要知道：平台不是按「你发了几个字」死板计费，而是按自己的规则拆成很多小段来算钱或算额度。免费额度用完，往往就是「这个月或这次试用能用的量用完了」，不是说你电脑坏了。

3. 密钥 / API Key（像一把临时门禁卡）
注册以后平台会给你一串密码一样的东西。谁拿到这串东西，谁就能冒充你的账号去用额度、花你的钱。所以：不要把它发在群里、不要截图带密钥、不要贴在公开网页上。后面说的「进仓库」「泄露」，指的就是这把「门禁卡」被别人抄走了。

先分清你在薅的是哪一种（很多人在这里搞混）

第一种：送你一点「余额」
就像话费：账户里有一点美元或人民币（或等价点数），你每用一次 AI，就扣一点，扣完为止。有的平台扣得慢，你会觉得「怎么用了好久还在免费」；有的用得凶，一两天就没了。

第二种：不收你钱，但限制你「多快能用一次」
你可以把它想成游乐园免门票但限流：进去不要钱，但每分钟、每天只允许你玩几次。技术上常写成 RPM（每分钟几次）、RPD（每天几次）、TPM（每分钟能用多少字量）——你不需要背英文，只要知道：不是坏了，是人家怕你一口气把免费资源打爆。如果你看到报错里写着 429，多半就是「太快了，等一下再试」。

第三种：根本不是给网站用的那种「接口」
比如 Cursor、GitHub Copilot、Codeium 这类，是装在写代码的软件里的助手。你在软件里跟 AI 聊天、让它改文件，用的是这个软件套餐里的次数或时长。

这跟「我做一个网站，别人打开我的网页，点一下按钮，让我的服务器去叫 AI，再把结果给访客看」完全是两回事。
用人话说：你在编辑器里用得爽，只代表你个人写东西方便；不代表你的网站已经自动接好了 AI，也不代表你的网站访客能免费蹭到同一份额度。 要做网站、做产品给陌生人用，一般要再走「第一种或第二种」那种正规 API，该绑卡绑卡，该限额限额。

免绑卡或几乎不用绑信用卡（适合先伸手试试）

Google AI Studio / Gemini
用 Google 账号打开 AI Studio，按提示申请一把 API Key（就是上面说的「门禁卡」）。我印象里很多时候不用绑信用卡就能玩，但以你注册那一刻网页上写的为准。
它家 Flash 系列往往一次能塞进去很长的材料：一大段文字、日志、PDF、截图，想试试「一口气丢给 AI」可以玩这个；更厉害的 Pro 在免费档往往卡得更严。
另外：免费档你的内容会不会被拿去训练、能不能拿来做生意，要看用户协议，别只看别人总结的「二手结论」。

Groq
注册以后一般就能调，具体能多用快、每天用多少，看 官方 rate limits。
它家给人的直观感受往往是快：你一点，字很快就一行行出来，适合你想做一个「像聊天一样打字出来」的小演示。上面多是 Llama 一类偏开源路线的模型，别拿它跟各家最贵的、不公开细节的模型比「谁更聪明」——你可以把它当成：先免费把流程跑顺、把界面做漂亮，再决定要不要花钱换更强的。

Hugging Face
免费注册能试很多模型，但要排队，有时候也不稳定，适合学生、爱好者学习、对照教程，不适合当成「我唯一依赖的生产线」。

国内各种「兼容 OpenAI」的第三方
很多会送一点注册金，把连接地址改成它们提供的就行（懂技术的人会改一个叫 base_url 的配置）。
对非技术读者的提醒：不要把重要账号的密钥随便填进你不认识的小网站。真有那种情况：别人后台能看到你调用了什么、用了多少——轻则隐私难看，重则额度被人滥用。正经生意尽量走官方或你信得过的大供应商。

一般要绑信用卡才会给「试用金」

绑卡不等于马上扣款，但很多平台是「先登记一张卡，才给你试用额度」。一定要看清楚：试用期过了、额度用完了之后，会不会自动按正价扣钱。不懂就宁可先问客服、先看帮助页，别闭着眼睛点「下一步」。

OpenAI
新账号过去常会送一点试用额度，具体送多少、送多久，以你账户里的账单/余额页面为准，别信网上互相抄的数字。还有面向研究、创业等的申请，额度更大，材料也更麻烦，需要的话去 help.openai.com 自己翻。
文档多、例子多，很多程序员第一次接 AI 会从这里抄示例；里面的 chat/completions 可以理解为一种标准的「点菜句式」，你不需要现在就懂，只要知道「这是人家文档里的常用叫法」。

Anthropic（Claude）
在 Console 注册后，不少账号会有一点试用金（多少、何时过期看账户页）。很多人感觉它写长文、改稿子、听话程度不错，但计费和 OpenAI 不是同一套规则。别一上来就疯狂点、开很多窗口同时刷，试用金可能瞬间烧光。

DeepSeek
国内手机号等注册，在控制台创建密钥。新用户送 token 或余额的活动有过，以当下控制台为准。中文写需求、补代码，很多人觉得很划算。官方会调价、会限流、会下线某个旧名字——如果你以后找人做网站，记得提醒对方：别把「模型名字」和「单价」写死在程序里好几年不更新。

云厂商送的几百美元试用券（门槛更高，别当「白送零花钱」）

像 GCP、AWS、Azure 这类大公司，新用户有时会给带期限的抵扣金，一般必须绑信用卡。用超了、或者试用期结束后资源没关，会真扣钱，而且不一定是一笔小钱。

它们可以把多家 AI 挂在同一张「云账单」上，适合公司、团队要考虑内网、权限、审计的时候。
普通人最容易踩的坑是：以为免费期一过就什么都不扣了，其实机器还在空转、服务还在后台跑，下个月信用卡账单出来才傻眼。所以人家提供的「预算告警」（budget alert，就是快花钱了给你发邮件提醒）——该开就开，不是摆设。

写代码软件里自带的 AI（Cursor / Copilot / Codeium 等）

你可以把它们想成：Word 里自带的助手，只不过对象是「写程序的文件」。一般是免费档加一点试用，按月算次数，或者某些高级功能单独限。

再强调一遍跟前面的区别：

  • 这些：主要服务你自己在这个软件里干活。
  • 网站 / App 给陌生人用：通常要在你自己的服务器（或云函数）上接官方或正规供应商的 AI，再把你网站用户的提问转过去。

不能把「我买 Copilot 了」理解成「我网站的访客也能无限免费用同一个 AI」——两套收费、两套规则，中间没有自动桥梁。

我踩过或看别人踩过的坑

试用金烧完以后还在扣钱
有人没关「自动续费 / 自动按量付费」，或者以为「余额没了就会自动停」，结果第二天请求还在发，按正价一笔笔扣。所以绑卡那天，把消费上限、邮件/短信提醒能开的都开；别等银行短信来了才第一次打开账单。

密钥泄露（门禁卡被别人抄走）
不懂技术的读者只要记住：那一长串 Key，性质像银行卡密码。有人把它贴在公开网页、发在 GitHub 公开仓库、或者打包进手机 App 里，会被机器人扫到，几小时花掉惊人额度的新闻每年都有。
如果你雇人做网站，可以提醒对方：密钥放在服务器配置里，不要写进给用户下载的文件里。

一直点、一直刷，被封或一直报错
免费层往往限流。脚本拼命重试、或者用户狂点按钮，又不「等几秒再试」，轻则一直失败，重则账号被风控。看到类似 429 的提示，先慢下来。

写死「模型名字」
平台把某个试用型号下线或改名，程序里没改，可能直接报错，或者悄悄用到更贵的默认型号。外行可以这么理解：菜单换了，你还按旧菜名点菜，要么点不上，要么被默认成最贵的套餐。 做维护的人应该把「菜名」放在配置里，改起来方便。

只依赖一家小第三方
对方改价、改规则、证书过期甚至不干了，你的页面就全挂。自己玩可以，正经生意最好有备用方案。

云试用结束忘了关机器
为了演示开了一台很贵的云服务器，以为「我不用就不花钱」，其实很多是按开机时间算的。演示完：该关机关机关，该删删。

几个顺手习惯（给会动手的人，外行可以跳过）

每个项目用单独一把密钥，哪把泄露了只影响这一件事。隔一段时间打开「用量」或「账单」看一眼，你常会突然发现：一直是某一个选项在悄悄花钱——很多人第一次看账单才意识到自己一直在用最贵那一档。

还不知道选哪个？先这么试（主观，不保证最适合你）

想先做一个字一条条往外蹦的小演示，图个爽快：可以试试 Groq 和 Gemini Flash。
手里有一大段字、长文档，想试试谁吃得下：可以在 Gemini（看免费限额）和 Claude（看版本和限额）里同一段文字各试一次，哪个顺手用哪个。
第一次接「网站调 AI」、想对着说明书一步步抄：很多人从 OpenAI 或兼容 OpenAI 写法的正规供应商开始。
中文需求多、想省一点：DeepSeek 官方或你验证过的国内线路。
已经是公司行为、要讲合规、要进内网审计：云厂商试用券 + 预算告警，比到处攒零散密钥更像个正经工程。
