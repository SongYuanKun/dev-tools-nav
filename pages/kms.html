💻 KMS 激活

Windows / Office / Server 全版本 KMS 激活，以管理员身份运行 CMD 或 PowerShell 执行下列命令。

KMS 服务器：kms.jpy.wang

说明：先执行 slmgr /skms kms.jpy.wang 设置 KMS 服务器，再执行 slmgr /ato 激活。若需安装密钥，使用 slmgr /ipk <密钥> 后再执行上述两步。

Windows 11 / Windows 10 专业版 / 企业版

slmgr /skms kms.jpy.wang
slmgr /ato

Office 2024 / 2021 / 2019（需以管理员运行 CMD，并 cd 到 Office 安装目录）

cd %ProgramFiles%\Microsoft Office\Office16
cscript ospp.vbs /sethst:kms.jpy.wang
cscript ospp.vbs /act

Windows Server 2022 / 2019 / 2016

slmgr /skms kms.jpy.wang
slmgr /ato
